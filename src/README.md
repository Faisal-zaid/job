# Description

This is the frontend part of the project we are doing as a group, it is meant to help job seekers apply for jobs that fit there need and employers to post jobs to find employees.
It focuses on building the user interface using React and interacting with the backend logic.
Each feature is organized clearly to make the project easy to understand and follow.
